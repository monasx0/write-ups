# TryHackMe Write-ups

A collection of practical cybersecurity write-ups and solutions from TryHackMe rooms.

**Visit the site:** https://monasx0.github.io/write-ups/

## Overview

This repository contains detailed walkthrough guides for various TryHackMe challenges, covering topics like:
- Web exploitation
- Privilege escalation
- Reconnaissance and enumeration
- Cryptography and steganography
- And more

## Repository Structure

- `_posts/` - Write-up markdown files
- `assets/img/writeups/` - Images and screenshots
- `_config.yml` - Site configuration
- `.github/workflows/` - GitHub Pages deployment automation
